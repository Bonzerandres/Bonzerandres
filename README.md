- Designing analytics-ready models for financial, operational, and business data
- Automating data quality checks, lineage, documentation, and CI/CD workflows

- Turning messy business systems into reliable data platforms

## Main Projects

### Recession Signal Monitoring Platform
Built a pipeline that tracks U.S. recession risk using multiple FRED economic indicators, dbt models, DuckDB, MotherDuck, and dashboard-ready datasets.

### Agentic FinOps Intelligence Platform
Building a cost-intelligence platform that turns cloud spend, usage, and infrastructure data into analytics-ready models and AI-assisted recommendations.

### ATR Platform Unification
Designed a modernization plan for fragmented CRM, ERP, warehouse, support, e-commerce, and reporting systems into a unified operational data platform.

### Scarlet Visitor Management Platform
Built a security-focused visitor and building-access platform with identity validation, access controls, data integrity, and CI/CD delivery workflows.
